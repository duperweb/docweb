Estrikti yon dokiman html

yon dokiman html tankou, yon moun. li gen html tags ki se kò a, apre sa doctype la di navigatè ki tip dokiman w'ap ekri a, li gen head ki se tèt la. sa ki nan tèt la , li pa vizib. nan head la , lòt tags ki pral anndan l tou.

  • title: ki se tit la
  • link: se pou css ak lot
  • meta: gen anpil done

Tags - Etikèt oubyen baliz

Tags yo tankou pè soulye, tenis, oubyen sandal. rezon ki fè nou di sa, se paske menn jan nou pale de yon grenn soulye goch ak yon grenn dwat. tags yo html souvan mache de pa de. gen sa yo rele opening tag ki se komansman yon eleman e gen sa yo rele closing tags li met fen nan yon eleman. Yon tags se yon mo-kle ki ant de kwochè. <nontag>kontni - sa se teks ou lot eleman</nontag>

Element - Eleman

yon eleman nan html, se tags ak kontni li. nested element se eleman ka al anndan eleman. gen eleman ki kèk ka eleman yo konn pa sa nou te ka rele pitit. < br> ki se liy brize.

fè yon ti koudèy

block element e inline element

block element se yon eleman ki egzije eleman ki apre l la pran yon nouvo liy, tankou paragraf

inline element se eleman ki pa egzije eleman ki apre l pran yon nouvo liy.

Case Sensitivity - Kas Sansib

pou HTML tag yo ka an majiskil oubyen an miniskil, sa vle di si youn gen majiskil, lòt miniskil yo menm . li rekomandab pou n itilize, miniskil.
